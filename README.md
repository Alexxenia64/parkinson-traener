# Parkinson Træner [![Åbn App](https://img.shields.io/badge/GitHub%20Pages-Live-brightgreen)](https://alexoneta64.github.io/parkinson-tremor/)

En enkel webapp til daglig træning og registrering af dagsform for personer med Parkinson.

## Funktioner
- Vælg din dagsform: **God**, **Middel**, eller **Dårlig**
- Appen husker dit seneste valg via browserens localStorage
- Enkel og mobilvenlig brugerflade
- Kan tilføjes til startskærmen på Android/iOS

## Live version
👉 [Åbn Parkinson Træner her](https://alexoneta64.github.io/parkinson-tremor/)

## Teknologi
- HTML5  
- CSS3  
- JavaScript (ES6)  
- Hostet via GitHub Pages

## Fremtidige funktioner
- [Dagbog](ca://s?q=Tilfoej_dagbogsfunktion_til_parkinson_app): mulighed for at skrive korte noter om dagens oplevelse  
- [Statistik](ca://s?q=Tilfoej_statistik_til_parkinson_app): grafer over udvikling i dagsform over tid  
- [Eksport](ca://s?q=Tilfoej_CSV_eksport_til_parkinson_app): gem data som CSV til deling med læge eller forsker  
- [Træningsbibliotek](ca://s?q=Tilfoej_traeningsbibliotek_til_parkinson_app): forslag til Parkinson‑venlige øvelser  
- [Brugerlogin](ca://s?q=Tilfoej_brugerlogin_til_parkinson_app): personlig profil og sikker lagring af data
